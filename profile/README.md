## PLAB은 실습실 대여의 불편함을 해결하기 위해 노력하고 있습니다.

기존의 실습실 대여 시스템을 편하게 이용할 수 있도록 하는것을 목표로 두고 있습니다.

또한 단순히 개발에서 끝나는 것이 아닌 편리하게 사용할 수 있도록
조금씩 개선해 나가며 관리하고 있습니다.
